# 🐢 Curso de Logo y Python para Niños
## ¡Aprende a programar y haz que el ordenador obedezca tus órdenes!

**Edad recomendada:** 9-11 años
**Duración total:** 12 semanas (3 sesiones por semana, 45-60 minutos cada una)
**Nivel:** Principiante absoluto
**Idioma:** Español

---

## 👋 ¡Hola, futuro programador o programadora!

¿Alguna vez has querido hacer que el ordenador dibuje lo que tú quieras, o que resuelva problemas por ti? ¡Pues eso es exactamente lo que vas a aprender en este curso!

Vamos a aprender **dos lenguajes de programación**:
- 🐢 **Logo**: Para dibujar figuras increíbles con una tortuga virtual
- 🐍 **Python**: El lenguaje favorito de muchos científicos y programadores del mundo

¡Al final del curso serás capaz de crear tus propios programas!

## 📋 Consejos para aprender mejor

1. **No hay prisa** – Lee la teoría con calma antes de empezar a practicar
2. **Experimenta** – Cambia los números y letras para ver qué pasa
3. **Equivocarse es normal** – Los errores son parte del aprendizaje
4. **Pide ayuda** – Si algo no funciona, busca el error juntos con un adulto
5. **Diviértete** – ¡La programación es como un juego!
