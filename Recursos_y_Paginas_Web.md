## 💡 Conceptos para investigar cuando termines el curso

Una vez terminado este curso, podrías investigar:

1. **Turtle en Python** – Dibujar como en Logo pero con Python
   ```python
   import turtle
   t = turtle.Turtle()
   t.forward(100)
   t.right(90)
   ```

2. **Pygame** – Crear videojuegos con gráficos en Python

3. **HTML y CSS** – Crear páginas web

4. **JavaScript** – Programar en el navegador

5. **Scratch avanzado** – Crear juegos y animaciones

## 📌 Notas de instalación

### Windows 10/11:

**Python:**
1. Ir a python.org
2. Descargar Python 3.x
3. ✅ Marcar "Add Python to PATH"
4. Instalar y abrir CMD
5. Escribir `python --version` para verificar

**Thonny:**
1. Ir a thonny.org
2. Descargar instalador para Windows
3. Instalar normalmente
4. ¡Listo para programar!

**FMSLogo:**
1. Ir a fmslogo.sourceforge.net
2. Descargar la versión más reciente
3. Instalar y ejecutar
4. La tortuga estará lista
